How It Works

Set up in minutes. Research in seconds.

AI-Prowler sits between your files and Claude. It handles the indexing, embedding, and retrieval — Claude handles the thinking.

01

Index your documents

Point AI-Prowler at any folder on your PC. It extracts text from PDFs, Word docs, spreadsheets, emails, images (via OCR), and 65+ other formats, then builds a private ChromaDB vector database on your machine.

02

Ask Claude anything

Open Claude Desktop. Claude automatically discovers AI-Prowler’s 81 tools via MCP and uses them to search, expand, and reason over your documents — going as deep as needed to answer your question.

03

Access from any device

Subscribe for mobile access and your knowledge base travels with you. One activation code sets up a secure Cloudflare tunnel — use Claude.ai from your phone, tablet, or any browser to reach your documents.

FAQ

Common questions.

Do I need a Claude subscription to use AI-Prowler?
A free Claude account works with Claude Desktop for local use. For mobile access via Claude.ai, you need Claude Pro ($20/month from Anthropic) because custom MCP connectors require a paid Claude plan. Claude Pro is separate from the AI-Prowler subscription.
Does my data leave my computer?
Your documents, the vector database, and all indexed content stay on your PC. The only data that leaves is the text of retrieved document chunks when Claude processes them via the Anthropic API — the same as any Claude conversation. AI-Prowler never uploads your files to any server.
How does mobile access work technically?
When you subscribe, we provision a Cloudflare tunnel to your PC and send you an activation code by email. Enter the code in AI-Prowler Settings → Remote Access and click Configure — everything sets up automatically. Your PC then has a permanent URL that Claude.ai can reach securely from any device.
What is the difference between Personal and Business?
Personal gives one person mobile access to their own knowledge base on one PC. Business adds server mode — a shared team knowledge base on a company machine — plus a personal AI-Prowler install for each team member. Each Business seat gets both: their own personal knowledge base and access to the shared company one.
Does Business include server hardware or installation support?
No. AI-Prowler Business is software only — you provide and set up the team server yourself (or with your own IT support). If you'd like recommendations on server hardware for your team size, help sizing hardware for your specific application, or paid installation support, contact david.vavro1@gmail.com.
Can I cancel my subscription?
Yes, cancel anytime. Your tunnel stays active until the end of the billing period. After cancellation, the local AI-Prowler Desktop app continues to work fully — you just lose remote mobile access.
How do I get started?
Download AI-Prowler Desktop for free, install it, and add your first document folder. Open Claude Desktop — it connects to AI-Prowler automatically. For mobile access, click Subscribe in Settings → Remote Access — checkout and activation are fully automated, no email required. Business also has server hardware and installation support available — see the FAQ above.
